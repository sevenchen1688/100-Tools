# PDF Toolkit - 全功能 PDF 处理工具

## 1. 项目概述

**项目名称**: PDF Toolkit
**项目类型**: Python 桌面应用程序
**核心功能**: 提供全面的 PDF 处理功能，包括格式转换、内容编辑、安全加密等
**目标用户**: 办公人员、学生、教师等日常需要处理 PDF 文件的用户

## 2. 技术栈

| 组件 | 技术选型 | 说明 |
|------|---------|------|
| GUI 框架 | Tkinter | Python 内置，零依赖 |
| PDF 解析 | PyMuPDF (fitz) | 高性能 PDF 处理库 |
| PDF 操作 | pypdf | PDF 合并、拆分、加密等 |
| Word 转换 | python-docx | 生成 Word 文档 |
| 图片处理 | Pillow | 图片格式转换和处理 |
| 打包工具 | PyInstaller | 生成可执行文件 |

## 3. 功能清单

### 3.1 格式转换
- ✅ **PDF 转 Word** - 将 PDF 内容转换为可编辑的 Word 文档
- ✅ **PDF 转图片** - 将 PDF 每一页转换为图片（JPG/PNG）
- ✅ **图片转 PDF** - 将多张图片合并为一个 PDF 文件
- ✅ **Word 转 PDF** - 将 Word 文档转换为 PDF

### 3.2 PDF 编辑
- ✅ **去除水印** - 智能识别并去除 PDF 中的水印（图片/文字）
- ✅ **添加水印** - 为 PDF 添加文字或图片水印
- ✅ **PDF 压缩** - 压缩 PDF 文件大小
- ✅ **PDF 加密** - 为 PDF 添加密码保护
- ✅ **PDF 解密** - 移除 PDF 密码保护

### 3.3 PDF 处理
- ✅ **PDF 合并** - 将多个 PDF 文件合并为一个
- ✅ **PDF 拆分** - 按页面范围拆分 PDF
- ✅ **PDF 旋转** - 旋转 PDF 页面（90°/180°/270°）
- ✅ **提取页面** - 从 PDF 中提取指定页面
- ✅ **删除页面** - 从 PDF 中删除指定页面
- ✅ **提取文本** - 提取 PDF 中的文本内容
- ✅ **提取图片** - 提取 PDF 中的所有图片

### 3.4 PDF 信息
- ✅ **PDF 预览** - 显示 PDF 缩略图预览
- ✅ **PDF 信息** - 显示 PDF 文件信息（页数、大小、作者等）

## 4. 功能详细说明

### 4.1 PDF 转 Word
- 保留原始排版和格式
- 支持文本、表格、图片提取
- 支持中英文混排
- 可选择转换特定页面范围

### 4.2 PDF 转图片
- 支持 JPG/PNG 格式输出
- 可调节图片质量（DPI）
- 支持批量转换
- 可选择特定页面转换
- 支持保持原始尺寸或自定义尺寸

### 4.3 去除水印
- 智能识别水印类型（图片水印/文字水印）
- 支持全文档或指定页面
- 保留原始内容完整性
- 提供预览对比功能

### 4.4 其他功能
- **PDF 合并**: 支持拖拽排序，批量合并
- **PDF 拆分**: 支持按页面范围、每 N 页拆分
- **PDF 压缩**: 支持不同压缩级别（高/中/低）
- **PDF 加密**: 支持设置打开密码和权限密码

## 7. 错误处理

### 7.1 常见错误
| 错误类型 | 处理方式 | 用户提示 |
|---------|---------|---------|
| 文件不存在 | 提示选择文件 | "文件不存在，请重新选择" |
| 文件格式错误 | 提示正确格式 | "请选择 PDF 文件" |
| 文件被占用 | 提示关闭文件 | "文件正在被其他程序使用" |
| 转换失败 | 记录日志并提示 | "转换失败: [原因]" |
| 内存不足 | 提示分批处理 | "文件较大，请分批处理" |

### 7.2 日志记录
- 记录所有操作日志
- 记录错误详细信息
- 支持日志文件查看
- 按日期分割日志

## 8. 性能优化

- 使用多线程处理，避免界面卡顿
- 大文件分批处理
- 内存使用监控
- 进度实时更新
- 缓存机制减少重复计算
