# PDF Toolkit - 全功能 PDF 处理工具

一款功能强大的桌面应用程序，支持 PDF 格式转换、内容编辑、安全加密等全面的 PDF 处理功能。

## 📋 功能概览

### 格式转换
- **PDF 转图片** - 将 PDF 每一页转换为高清图片（PNG/JPG）
- **图片转 PDF** - 将多张图片合并为一个 PDF 文件
- **PDF 转 Word** - 将 PDF 内容转换为可编辑的 Word 文档
- **Word 转 PDF** - 将 Word 文档转换为 PDF

### PDF 编辑
- **去除水印** - 智能识别并去除 PDF 中的水印
- **添加水印** - 为 PDF 添加文字或图片水印
- **PDF 压缩** - 压缩 PDF 文件大小
- **PDF 加密** - 为 PDF 添加密码保护
- **PDF 解密** - 移除 PDF 密码保护

### PDF 处理
- **PDF 合并** - 将多个 PDF 文件合并为一个
- **PDF 拆分** - 按页面范围或每 N 页拆分 PDF
- **PDF 旋转** - 旋转 PDF 页面（90°/180°/270°）
- **提取页面** - 从 PDF 中提取指定页面
- **删除页面** - 从 PDF 中删除指定页面
- **提取文本** - 提取 PDF 中的文本内容
- **提取图片** - 提取 PDF 中的所有图片

## 🖥️ 运行环境

- Python 3.8 或更高版本
- Windows 7/8/10/11 或 macOS 10.14+
- 最低 4GB 内存
- 约 100MB 硬盘空间

## 📦 安装步骤

### 方式一：使用源码

1. 确保已安装 Python 3.8+ 环境
   ```bash
   python --version
   ```

2. 进入项目目录：
   ```bash
   cd PdfTools
   ```

3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

4. 启动程序：
   ```bash
   python main.py
   ```

### 方式二：使用可执行文件

直接下载并运行打包好的可执行文件（如果可用）。

## 🚀 快速开始

### 基本使用流程

1. **启动程序**
   运行 `python main.py` 或双击可执行文件

2. **选择功能**
   在左侧功能列表中点击要使用的功能

3. **添加文件**
   - 点击"添加文件"按钮选择 PDF 文件
   - 或直接将文件拖拽到文件列表区域
   - 支持批量添加多个文件

4. **设置选项**
   根据所选功能，在选项区域设置相应参数

5. **选择输出目录**
   点击"浏览..."按钮选择输出位置

6. **开始处理**
   点击"开始处理"按钮，等待处理完成

## ❓ 常见问题

### Q: 转换后的文件在哪里？
A: 转换后的文件保存在您选择的输出目录中，可在状态栏查看具体路径。

### Q: 支持批量处理吗？
A: 支持！可以一次添加多个文件，程序会自动按顺序处理。

### Q: 处理大文件时很慢怎么办？
A: 程序采用多线程处理，可以尝试：
- 降低图片转换的 DPI
- 使用分批处理
- 确保有足够的内存

### Q: 去水印功能不生效怎么办？
A: 部分复杂水印可能无法完全去除，建议：
- 尝试不同的水印类型选项
- 对于重要文档，建议联系原文件所有者

### Q: 忘记了 PDF 密码怎么办？
A: 很抱歉，解密功能需要正确的密码才能操作。请尝试联系文件所有者获取密码。

## 🔒 安全说明

- 所有处理操作均在本地完成，不会上传任何文件
- 不收集任何用户数据
- 日志文件仅记录操作信息，不包含敏感内容
- 建议设置强密码保护重要 PDF 文件
